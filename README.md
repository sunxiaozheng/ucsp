# ucsp
The University Class Scheduling Problem

PHP写的一个简易版的排课系统

| 路由        | 功能   |
| --------   | -----:  |
| coursediv     | 教师当天的课分散排列 |
| chksubjmutex        |   科目互斥   |
| amnotnexttopm        |    上午末节下午首节不能为同一个老师    |
| coursenotnextto        |    禁止科目相邻    |
| lmtcoursebysubj        |    限制学科本周的上课节数    |
| lmttechdailycoursenum        |    限制教师当天的上课节数    |
